# Ergebnis-Bilder (Grid, mittlerer Abschnitt)

Lege hier deine **6 Ergebnis-Bilder** ab (per Drag & Drop über die GitHub-Weboberfläche
oder lokal hinzufügen und pushen). Sie werden automatisch als Raster (2 Spalten) angezeigt.

- Unterstützte Formate: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`
- Reihenfolge = alphabetisch/numerisch nach Dateiname → benenne sie z. B. `1.jpg` … `6.jpg`.
- Erkennung erfolgt **automatisch** über die GitHub-API – kein Code-Eingriff nötig.
- Die `result-*.svg` sind nur Platzhalter – einfach löschen, sobald du eigene Bilder hast.
- Hochformat-Bilder (Ergebnislisten) werden vollständig angezeigt (nicht beschnitten).

## Fallback (optional)
Falls die GitHub-API mal nicht erreichbar ist, nutzt die Seite `images.json` als Reserve.
Für den Normalbetrieb ist das **nicht** nötig.
